# Gestion de location de véhicules – PL/SQL

## Description
Projet universitaire de gestion de location de véhicules 
développé en PL/SQL sous Oracle.

## Fonctionnalités
- Modélisation de la base de données relationnelle
- Procédures stockées et fonctions
- Triggers d'automatisation des règles métier
- Gestion des transactions et contraintes d'intégrité

## Technologies utilisées
- Oracle Database
- PL/SQL

## Contexte
Projet réalisé en L3 Informatique – Université de Lorraine (2025-2026)
